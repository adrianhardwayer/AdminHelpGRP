deadcmd_view.lua
Это lua-скрипт подсказка который работает с встроенными библиотеками и помогает тебе быстро узнать правила проекта и команды которые тебе доступны. Является быстрой подсказкой для большинства обывателей и защитников сервера.

Установка:
Взять файлы resource и deadcmd_view.lua и поместить в папку Moonloader.
Включается на F12.

Чтобы поменять фотографию или подсказку, просто замени изображение в moonloader/resource/deadcmd_view/view.png* 1920 на 1080.
Все вопросы vk.com/deadcmd, предложения или поддержка.
